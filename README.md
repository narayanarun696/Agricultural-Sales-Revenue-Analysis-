# Agricultural-Sales-Revenue-Analysis-
A fully automated, end-to-end data solution for agricultural sales and revenue analysis — from raw data ingestion via Azure Data Factory to a live Power BI dashboard with scheduled refresh and trigger-based automation.

This project demonstrates a production-style data pipeline for the agriculture sector — ingesting raw sales data, transforming it through ADF, loading it into Azure SQL, and surfacing it in a Power BI dashboard that refreshes automatically when new data lands.

Key focus areas:
1.Automated daily ingestion of agricultural sales & revenue data
2.Data transformation and cleaning via ADF Data Flows
3.Event-triggered and scheduled pipeline execution
4.Live Power BI dashboard with gateway-based auto-refresh
5.KPI tracking: revenue, sales volume, regional performance
